# 如何使用

## 从源代码构建

使用cmake

或者[点击这里](https://github.com/NaviHX/hzip/releases/tag/1.0.1)下载windows下的二进制文件

## 用法

```
Usage          : hzip [options] file_path 
-h, --help     : help information 
-c, --compress : compress a file 
-e, --extract  : extract a file 
```
